Пояснительная записка

Данный веб-ресурс был разработан для личного пользования. Его задача - возможность предлагать пользователям выбирать товары и услуги и заказывать их.

Была проведена следующая работа:
1. Дизайн и отрисовка макета сайта в figma.
2. Верстка сайта на языке разметки html (главная страница, страница с каталогом, страница с контакнтами и мини-игра)
3. Добавление стилей страницам сайта.
4. Адаптация сайта для ноутбуков, планшетов, мобильных устройств.
5. Написание кода на языке JavaScript :

1. Реализация скрывающегося меню при изменении размера экрана.
2. Добавление в каталог товаров слайдера со сменяющимися картинками и возможностью самостоятельно переключать эти картинки.
3. Реализация корзины и возможности добавлять в нее товары, изменять их количество и удалять товары.
4. Добавлена интерактивная карта Яндекс на странице "Контакты".
5. Реализация игры "Морской бой"

Сайт тестировался и работает в следующих браузерах: 1. Google Chrome 2. Mozilla Firefox 3. Opera 5. Yandex browser

При создании сайта, были использованы следящие ПО: 1.figma 2.Visual Studio Code 3. Графиеские редакторы для работы с изображениями

Структура сайта

На главной странице отображаются виды предоставляемых услуг и возможность заказать услугу. Страница с каталогом представляет собой список возможных товаров и возможность положить выбранный товар в корзину. Страница с мини-игрой, на котрой можно методом ввода координат попасть в корабли, или промазать. Когда все корабли потоплены, пользователю сообщается количество попыток, за которые он потопил корабли. Страница с контактами и интерактивной картой, а так же анимированным фоновым изображением. Страница с карточкой товара и галереей, написанной на языке JS. Cтраница с формой заказа услуги.
